Opera

Opera grundades 1995 av det norska företaget Opera Software ASA och startade som en forskningsprojekt vid företaget Telenor. Opera har utmärkt sig genom att vara pionjärer inom flera områden. Opera skiljer sig från andra webbläsare genom att de från allra första början tog betalt för att använda tjänsten. Detta ändsades sedan och Opera är nu gratis.

2016 uppköpta av ett Kinesiskt företag (kunhoo software llc).

Vad som skiljer Opera från andra webbläsare:

  • Opera turbo (Turbo mode krymper webbplatsernas innehåll innan de laddas ned. Ex. så krymps bilder och videos till rätt dimensioner (för t.ex telefoner) innan de laddas ned. Det fungerar genom att sidorna passerar andra servrar som identifierar vad som kan tas bort.
  • Speed dial (snabbval)
  • Batterisparande webbsida som kan spara upp till en timme i batteritid. Ikonen finns vid sök - och adressfältet som du ser när du kopplar bort din laddare. Klicka på den för att aktivera batterispararen. När datorn går ner till 20 procent i batteritid så talar den om det och ber dig att sätta på batterispararen för att gå in i sparläge.
